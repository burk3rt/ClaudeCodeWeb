# Project Context

## AI-Supported Learning Platform

We are developing an AI-supported learning platform with the following infrastructure:

- **Kubernetes**: k3s v1.33.5 (with pre-installed Traefik)
- **Authentication**: Keycloak
- **Authorization**: OpenFGA
- **Target Audience**: Students (Single-Tenant)
- **Performance**: Low requirements
- **AI Integration**: External LLMs for flashcard/quiz generation
